# `c-ui.chat.message`

- Tag: `<c-ui.chat.message />`
- Source: `bloomerp/django_bloomerp/bloomerp/templates/cotton/ui/chat/message.html`

## Description

Chat message component.

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `avatar` | - | Avatar image URL. If omitted, initials are shown. |
| `sender` | - | Sender display name |
| `message` | - | Message text |
| `datetime` | - | Message datetime. Rendered as a relative time. |
| `side` | - | Message side, "left" or "right". Default is "left". |
| `background` | - | Bubble background, "primary" or "secondary". Default is "secondary". |
| `variant` | - | Alias for background. |
| `class` | - | Additional classes for the message row |
