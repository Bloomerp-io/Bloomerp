# `c-ui.breadcrumb.item`

- Tag: `<c-ui.breadcrumb.item />`
- Source: `bloomerp/django_bloomerp/bloomerp/templates/cotton/ui/breadcrumb/item.html`

## Description

Breadcrumb Item Component

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `text` | - | The text to display for the breadcrumb item |
| `href` | - | URL for the breadcrumb link (optional) |
| `hx_get` | - | HTMX GET URL (optional) |
| `hx_target` | - | HTMX target element (optional) |
| `hx_swap` | - | HTMX swap method (optional) |
| `hx_push_url` | - | Whether to push URL to history (optional) |
| `icon` | - | Icon class to display before text (optional) |
| `active` | - | Whether this is the current/active item (default: False) |
| `class` | - | Additional CSS classes for the item |
| `show_separator` | - | Show separator after this item (default: True) |
| `separator` | - | Type of separator - 'chevron', 'slash', 'arrow' (default: 'chevron') |
