# `c-ui.steps`

- Tag: `<c-ui.steps />`
- Source: `bloomerp/django_bloomerp/bloomerp/templates/cotton/ui/steps.html`

## Description

The stepper component visually represents a multi-step process, indicating the current step and progress through the steps.

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `class` | str | Additional CSS classes for the container. |
| `current_step` | int | The current active step in the process (1-indexed). |
| `steps` | list | A list of step titles. |
