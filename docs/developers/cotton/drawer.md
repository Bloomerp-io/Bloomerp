# `c-ui.drawer`

- Tag: `<c-ui.drawer />`
- Source: `bloomerp/django_bloomerp/bloomerp/templates/cotton/ui/drawer.html`
- TypeScript component id: `drawer`

## Description

Drawer component.

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `id` | - | unique identifier for the drawer (required) |
| `side` | - | drawer side - "left", "right" (default: "right") |
| `size` | - | drawer size - "sm", "md", "lg", "xl", "full" (default: "md") |
| `backdrop` | - | whether to show the modal-style backdrop (default: True) |
| `backdrop_click_close` | - | close drawer when clicking backdrop (default: True) |
| `slot` | - | main content of the drawer |
